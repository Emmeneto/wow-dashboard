# WoW Dashboard

A smart weekly gearing tracker and Best-in-Slot companion for World of Warcraft: Midnight Season 1.

## Features

**In-Game Tracker**
- Compact, draggable frame showing your weekly gearing checklist
- Smart "Next Up" recommendations telling you the fastest path to max gear
- Auto-detects vault progress, quest completion, and gear changes in real-time
- TomTom waypoint integration — click any task to get directions
- Collapsible sections with vault slot progress tracking

**Weekly Gearing Priorities**
- Great Vault tracking for Dungeons, Raids, and World Activities
- Liadrin Spark Quest detection with objective progress (all 13 weekly variants)
- Nightmare Prey Hunt tracking with unlock guidance
- World Boss and Housing Weekly tracking
- Manual toggle for anything auto-detection misses

**Smart Recommendations**
- Priority engine based on the fastest Midnight Season 1 gearing path
- Context-aware advice that adapts to your current item level
- Tells you exactly what to do next and where to go

**Slash Commands**
- `/db` — Toggle the tracker
- `/db next` — Navigate to your next priority task
- `/db go` — Set a waypoint for your next task
- `/db mark worldboss` — Manually mark tasks as done
- `/db help` — Full command list

## Optional Companion App

An optional desktop companion app provides:
- Browser-based dashboard with character tiles and detailed stats
- Best-in-Slot gear comparison per slot with Wowhead data
- Tier set tracking (Voidbreaker's Accordance 2pc/4pc)
- Time estimates for remaining weekly tasks
- Multi-character support across all your alts

The companion reads your local SavedVariables — no data is sent to external servers. Fully compliant with Blizzard's addon policy.

## Installation

1. Download and extract to your `Interface/AddOns/` folder
2. `/reload` in game
3. The tracker appears automatically on level 90 characters
4. Type `/db help` for all commands
